Happy DSA Day by Day 2!

Let's solve some DSA problems together!

Question 1: Rotate Image

What we learn: Matrix manipulation, in-place transformations, and rotation using matrix transposition and row reversal.

Question 2: Spiral Matrix

What we learn: Traversing a matrix in a spiral order, boundary conditions, and handling matrix edges in a 2D array.

Question 3: Majority Element

What we learn: The Boyer-Moore Voting Algorithm, efficient counting techniques, and linear time solutions for finding a majority element.

Question 4: Subarray Sum Equals K

What we learn: Hash maps for tracking prefix sums, efficient ways to calculate subarray sums, and handling target-based counting in arrays.

Question 5: Subarray Product Less Than K

What we learn: Sliding window approach for variable-length subarrays, handling product calculations, and managing subarray boundaries dynamically.
