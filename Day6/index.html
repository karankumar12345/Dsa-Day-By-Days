Question 1: Sum of Subarray Minimums

What we learn:

  • Understanding how to calculate the contribution of elements in subarrays.
  • Using monotonic stacks to find the nearest smaller elements efficiently.
  • Optimizing brute force solutions for array-based problems.

Challenge: Implement the optimized approach with \(O(n)\) complexity.

Question 2: Queue using Two Stacks

What we learn:

  • How to implement a queue using stack operations.
  • Understanding the concept of amortized complexity.
  • Practicing data structure transformation and adaptability.

Challenge: Ensure all operations (enqueue, dequeue) are efficient and handle edge cases.
